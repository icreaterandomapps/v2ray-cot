### English

🚀 **Powerful V2Ray/Xray Configurations for Iran**

A comprehensive collection of high-performance V2Ray and Xray configurations designed to provide fast, stable, and secure connectivity for users in Iran. This repository includes carefully selected and regularly updated configurations based on modern protocols such as **VLESS**, **VMess**, **Trojan**, and **Reality**.

✨ Features:

* High-speed and low-latency configurations
* Stable connections optimized for challenging network conditions
* Support for VLESS, VMess, Trojan, and Reality protocols
* Regularly updated subscription links and configuration files
* Easy-to-use setup for multiple clients and platforms
* Community-driven improvements and contributions

Whether you're looking for reliable daily use, testing different protocols, or exploring advanced Xray configurations, this repository aims to provide a well-maintained and organized collection of useful resources.

مجموعه‌ای کامل از کانفیگ‌های پرسرعت، پایدار و به‌روز V2Ray و Xray که با هدف ارائه اتصال بهتر برای کاربران ایرانی گردآوری شده‌اند. این مخزن شامل کانفیگ‌های منتخب و بهینه‌شده مبتنی بر پروتکل‌های مدرن **VLESS**، **VMess**، **Trojan** و **Reality** است.

✨ ویژگی‌ها:

* کانفیگ‌های پرسرعت با تأخیر پایین
* پایداری بالا در شرایط مختلف شبکه
* پشتیبانی از پروتکل‌های VLESS، VMess، Trojan و Reality
* به‌روزرسانی منظم لینک‌های اشتراک و فایل‌های کانفیگ
* راه‌اندازی آسان در کلاینت‌ها و سیستم‌عامل‌های مختلف
